git --version

This command sets the author name and email address to be used with your commits. Git config is also used in other scenarios.
Or you can directly change in file by this command
git config --global --edit


create repository, means the project which can be access from the git. So to make folder git repo write below command.
git init
this will create a empty and hidden repository (.git) in the project folder


git status
this will show you the status of file, kuch change hua ho etc.


git add
this will use to add files/folder
git add -A
this will use to add all files/folder together


Now ab agr mene kisi file mai change kiya hai toh first mujhe us file ko add krna pdega using git add command than fir usko commit krenge


git commit -m " Commit Message"
This command changes the head. It records or snapshots the file permanently in the version history with a message. Jo change kiya ho vo message mai likh do (-m means message)


git log
This command is used to check the commit history. By default, if no argument passed, Git log shows the most recent commits first. We can limit the number of log entries displayed by passing a number as an option, such as -3 to show only the last three entries.
